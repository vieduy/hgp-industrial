# Partner logos

Drop the real partner logo image files here. The site references them by these
exact filenames (see `backend/app/content.py` → `PARTNERS`):

- `jotun.png`
- `akzonobel.png`
- `ppg.png`
- `nippon.png`
- `seamaster.png`
- `samhwa.png`
- `desam.png`
- `kcc.png`

Recommended: transparent PNG or SVG, ~200px wide. Until a file exists, the site
falls back to showing the brand name as a styled wordmark.
